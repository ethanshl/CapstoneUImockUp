Help Page:

Below you can find help and FAQs for all the questions you may have about the site.


Creating New User:


An advisor can sign-up to use this site by entering some credentials. This will include their full name, TU E-Mail address, department, and other important information.


Logging In:

When logging in to the advisor portal simple information such as a TU-Email address, and a password will be necessary to access the site. A person without these credentials will not be able to enter the site. All credentials that are entered will be encrypted for security purposes.


Home Page:

The home page is the inital landing page that will be accessed when a user enters the site after successfully logging in. This is where they will see a quick view of all the applications they have waiting for them. There is a number of pending applications attached to them, as well as the ability to press that button and see a quick gridview list of all the applications waiting.
